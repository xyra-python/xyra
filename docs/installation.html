Installation

You can install Xyra using pip, which is the recommended way.

Installing with pip

To install Xyra, run the following command in your terminal:


pip install xyra

This will install the latest stable version of Xyra along with its dependencies.

Installing from Source

If you want to get the latest development version or contribute to the project, you can install Xyra from source.


git clone https://github.com/xyra-python/xyra.git
cd xyra
pip install -e .

This will install the framework in editable mode, meaning any changes you make to the source code will be reflected immediately.

Verifying the Installation

To verify that Xyra was installed correctly, you can run the following command:


python -m xyra --version

This should print the installed version of the Xyra framework.
