Static Files

Xyra can serve static files such as CSS, JavaScript, and images. This is useful for serving the assets required by your web application's frontend.

Configuring Static Files

You can configure a directory for static files when you create the App instance.


from xyra import App

app = App(static_directory="static")

Any files in the static directory will be served automatically. For example, a file at static/css/style.css will be available at /css/style.css.

Custom Static Routes

You can also define custom routes for serving static files from different directories using the app.static() method.


from xyra import App

app = App()

# Serve files from the "assets" directory under the "/assets" URL path
app.static("/assets", "assets")

With this configuration, a file at assets/img/logo.png will be accessible at /assets/img/logo.png.
