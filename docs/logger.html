Logger

Xyra includes a simple logger to provide insights into your application's behavior. The logger can be enabled when you start the server.

Enabling the Logger

To enable the logger, set the logger parameter to True in the app.listen() method.


from xyra import App

app = App()

@app.get("/")
def home(req, res):
    res.text("Hello, World!")

if __name__ == "__main__":
    app.listen(8000, logger=True)
