Examples

This section provides complete examples of Xyra applications to help you get started.

Hello, World!

A minimal Xyra application.


from xyra import App, Request, Response

app = App()

@app.get("/")
def home(req: Request, res: Response):
    res.json({"message": "Hello, World!"})

if __name__ == "__main__":
    app.listen(8000)

Simple TODO API

A simple in-memory TODO list API.


import asyncio
from xyra import App, Request, Response

app = App()

# In-memory database
todos = []

@app.get("/todos")
def get_todos(req: Request, res: Response):
    res.json(todos)

@app.post("/todos")
async def create_todo(req: Request, res: Response):
    data = await req.json()
    todos.append(data)
    res.status(201).json(data)

@app.get("/todos/{todo_id}")
def get_todo(req: Request, res: Response):
    todo_id = int(req.params.get("todo_id"))
    todo = next((t for t in todos if t["id"] == todo_id), None)
    if todo:
        res.json(todo)
    else:
        res.status(404).json({"error": "Todo not found"})

if __name__ == "__main__":
    app.listen(8000)

WebSocket Chat

A simple real-time chat application using WebSockets.


import asyncio
from xyra import App, WebSocket

app = App()
connections = set()

@app.ws("/ws")
async def chat_socket(ws: WebSocket):
    await ws.accept()
    connections.add(ws)
    try:
        while True:
            message = await ws.receive_text()
            for connection in connections:
                await connection.send_text(message)
    except Exception:
        pass
    finally:
        connections.remove(ws)

if __name__ == "__main__":
    app.listen(8000)
