Templating

Xyra uses the Jinja2 templating engine to render HTML templates. This allows you to create dynamic web pages by embedding Python-like expressions and logic in your HTML files.

Configuring Templating

To enable templating, you need to specify the directory where your templates are stored when you create the App instance.


from xyra import App

app = App(templates_directory="templates")

Rendering Templates

Use the res.render() method to render a template. You can pass context variables to the template as keyword arguments.


from xyra import App, Request, Response

app = App(templates_directory="templates")

@app.get("/")
def home(req: Request, res: Response):
    res.render("index.html", title="Home", message="Welcome to Xyra!")

Template Example

Here's an example of a Jinja2 template (templates/index.html):


<!DOCTYPE html>
<html>
<head>
    <title>{{ title }}</title>
</head>
<body>
    <h1>{{ message }}</h1>
    <p>This is a dynamic page rendered by Xyra.</p>
</body>
</html>

When the / route is accessed, Xyra will render this template and substitute {{ title }} and {{ message }} with the values passed to res.render().
